# Script bash utilizando o apache cgi

## Url
```
http://localhost:8080/main.cgi
```



## Instruções de utilização

**Clonar repo**:
```
git clone

```


**Buildar e subir container:**
```
docker-compose up -d 
```

**Porta utilizada:**
```
 8080->80
```

**Apach config file**

```
 /config/httpd.conf 
```
